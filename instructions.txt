INSTALLATION
- Open Terminal (in your Utilities folder).
- Install Homebrew:
    ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
- Install Git:
    brew install git
- Install Homebrew Caskroom
    brew install caskroom/cask/brew-cask
- Install Sublime Text
    brew cask install sublime-text

DEVELOPMENT
- Open Terminal
- "ls" will list the files in the current folder you are in
- "cd" will allow you to change directories
- Start out in your home directory
    cd ~
- Then change directories to where your website lives
    cd Desktop/janaktull.github.io/
- Open your website in a web browser
    open index.html
- Edit the file using sublime text
    subl index.html
- Make changes to index.html, save, refresh the webpage
- When you're done making changes, back to Terminal
    git add .
    git commit -m "some useful message about what you changes"
    git push origin master
- Go to www.janaktull.com and you should see you're new website within a few seconds
